Introduction

The XSX Library is a feature-rich UI framework designed for Roblox cheat developers. It provides tools to create polished, interactive user interfaces with minimal effort. Key features include:

  • Draggable windows with smooth animations
  • Rainbow gradient effects for vibrant visuals
  • Customizable notifications and watermarks
  • Dynamic components like toggles, sliders, and dropdowns
  • Optimized for modern Roblox versions with bug fixes

Quick Start

Get started with XSX in just a few lines:

local XSX = loadstring(game:HttpGet('https://bauu.xyz/xsx.lua'))()
local Library = XSX:Init({title = "MyCheat", Key = Enum.KeyCode.Insert})
local Tab = Library:NewTab("Main")
Tab:NewButton("Test", function() print("Clicked!") end)

Setup

Integrate XSX by including the library in your client-side script, typically via a ModuleScript.

local XSX = loadstring(game:HttpGet('https://bauu.xyz/xsx.lua'))()
local Library = XSX:Init({
    title = "ExampleHook",
    company = "ExampleStudio",
    Key = Enum.KeyCode.Insert,
    RainbowEnabled = true,
    BlurEffect = true,
    Debug = false
})

After initialization, create tabs, components, and visual effects. The library handles UI toggling via the specified keybind.

Configuration

Customize XSX's appearance and behavior through the :Init() configuration table.

local Config = {
    title = "ExampleHook",
    company = "ExampleStudio",
    Key = Enum.KeyCode.Insert,
    RainbowEnabled = true,
    BlurEffect = true,
    BlurSize = 24,
    FieldOfView = 70,
    transparency = 0,
    backgroundColor = Color3.fromRGB(31, 31, 31),
    headerColor = Color3.fromRGB(255, 255, 255),
    companyColor = Color3.fromRGB(163, 151, 255),
    acientColor = Color3.fromRGB(167, 154, 121),
    darkGray = Color3.fromRGB(27, 27, 27),
    lightGray = Color3.fromRGB(48, 48, 48),
    Font = Enum.Font.Code,
    Debug = true,
    rainbowColors = ColorSequence.new{
        ColorSequenceKeypoint.new(0.00, Color3.fromRGB(241, 137, 53)),
        ColorSequenceKeypoint.new(0.33, Color3.fromRGB(241, 53, 106)),
        ColorSequenceKeypoint.new(0.66, Color3.fromRGB(133, 53, 241)),
        ColorSequenceKeypoint.new(1, Color3.fromRGB(53, 186, 241))
    }
}
Library:Init(Config)

Adjust these options to match your cheat's theme for a cohesive look.

Core Methods

:Init(config)

Initializes the library with the provided configuration.

ParameterTypeDescription
configtableConfiguration options
ReturnsLibrary instance

:Watermark(text)

Displays a bottom-screen watermark with animated fade-in and rainbow bar.

ParameterTypeDescription
textstringWatermark text
ReturnsWatermarkFunctions (Hide(), Show(), SetText(new), Remove())

:Notify(text, duration, type, callback)

Shows a notification with customizable type, duration, and callback.

ParameterTypeDescription
textstringNotification message
durationnumberDisplay time in seconds (default: 5)
typestring"notification", "alert", "error", "success"
callbackfunctionOptional callback on completion
ReturnsNotificationFunctions (SetText(new))

:NewTab(title)

Creates a new tab in the main UI window.

ParameterTypeDescription
titlestringTab name
ReturnsTabFunctions

:BeginIntroduction()

Initiates an animated introduction sequence displaying the company logo.

ParameterTypeDescription
Returnsself

:AddIntroductionMessage(message)

Adds a message to the introduction sequence with a fade-in effect.

ParameterTypeDescription
messagestringMessage text
ReturnsTextLabel instance

:EndIntroduction()

Concludes the introduction sequence with a fade-out animation.

ParameterTypeDescription
Returnsself

:Panic()

Resets all toggles and sliders to their default states.

ParameterTypeDescription
Returnsself

:SetKeybind(key)

Changes the keybind used to toggle the UI.

ParameterTypeDescription
keyEnum.KeyCodeNew keybind
Returnsself

:ShowUI(visible)

Toggles the UI's visibility with blur and FOV effects.

ParameterTypeDescription
visiblebooleanShow or hide UI
Returnsself

:Remove()

Destroys the UI and triggers :Panic().

ParameterTypeDescription
Returnsself

:GetUsername(), :GetUserId(), :GetPlaceId(), :GetJobId()

Utility methods to retrieve player and game information.

ParameterTypeDescription
Returnsstring or number

:Rejoin()

Rejoins the current game instance.

ParameterTypeDescription
Returnsnone

:Copy(input)

Copies text to the clipboard using Roblox clipboard functions.

ParameterTypeDescription
inputstringText to copy
Returnsnone

:UnlockFps(new)

Sets the FPS cap if supported by the Roblox environment.

ParameterTypeDescription
newnumberNew FPS cap
Returnsnone

Components

Components are created within tabs using :NewTab(). Each component returns a table of functions for manipulation.

NewLabel(text, alignment)

Creates a static text label with customizable alignment.

ParameterTypeDescription
textstringLabel text
alignmentstring"left", "center", "right" (default: "left")
ReturnsLabelFunctions (SetText(text), Remove(), Hide(), Show(), Align(new))

NewButton(text, callback)

Creates a clickable button with hover and click animations.

ParameterTypeDescription
textstringButton text
callbackfunctionFunction called on click
ReturnsButtonFunctions (Fire(), Hide(), Show(), SetText(text), Remove(), SetFunction(new))

NewSection(text)

Creates a section header for organizing components.

ParameterTypeDescription
textstringSection title
ReturnsSectionFunctions (SetText(new), Hide(), Show(), Remove())

NewToggle(text, default, callback)

Creates a toggle switch with an optional keybind.

ParameterTypeDescription
textstringToggle label
defaultbooleanInitial state (default: false)
callbackfunctionCalled with boolean state
ReturnsToggleFunctions (Toggle(), Remove(), Set(state), GetValue(), SetFunction(new), AddKeybind(default))

NewKeybind(text, default, callback)

Creates a keybind input for binding keys to actions.

ParameterTypeDescription
textstringKeybind label
defaultEnum.KeyCodeDefault key (default: Enum.KeyCode.P)
callbackfunctionCalled with key name
ReturnsKeybindFunctions (Fire(), SetFunction(new), SetKey(new), SetText(new), Hide(), Show())

NewTextbox(text, default, placeHolder, type, autoexec, autoclear, callback)

Creates a text input field with customizable size and behavior.

ParameterTypeDescription
textstringTextbox label
defaultstringDefault text
placeHolderstringPlaceholder text
typestring"small", "medium", "large" (default: "small")
autoexecbooleanExecute on input change (default: true)
autoclearbooleanClear on execution (default: false)
callbackfunctionCalled with input text
ReturnsTextboxFunctions (Input(new), Fire(), SetFunction(new), SetText(new), Hide(), Show(), Remove(), SetPlaceHolder(new))

NewSelector(text, default, list, callback)

Creates a dropdown menu for selecting options.

ParameterTypeDescription
textstringSelector label
defaultstringDefault selected option
listtableArray of option strings
callbackfunctionCalled with selected option
ReturnsSelectorFunctions (AddOption(new, callback), RemoveOption(option), SetFunction(new), Text(new), Hide(), Show(), Remove())

NewSlider(text, suffix, compare, compareSign, values, callback)

Creates a slider for numerical input with a draggable indicator.

ParameterTypeDescription

suffixstringText appended to value
comparebooleanDisplay max value (default: false)
compareSignstringSeparator for comparison (default: "/")
valuestable{min, max, default}
callbackfunctionCalled with current value
ReturnsSliderFunctions (Set(new, noCallback), Max(new), Min(new), SetFunction(new), GetValue(), SetText(new), Hide(), Show(), Remove())

NewSeperator()

Creates a horizontal separator for visual organization.

ParameterTypeDescription
ReturnsSeperatorFunctions (Hide(), Show(), Remove())

Visual Effects

:ApplyRainbow(instance, wave)

Applies a rainbow gradient effect to a UI element.

ParameterTypeDescription
instanceInstanceUI element to apply effect
wavebooleanUse animated gradient wave (default: false)
Returnsnone

Enable RainbowEnabled in the config to apply this effect to supported components.

Blur Effect

When BlurEffect = true, the UI applies a camera blur when opened, controlled by BlurSize and FieldOfView.

TweenWrapper

Manages animations with predefined styles.

TweenWrapper:CreateStyle("custom", 0.5, Enum.EasingStyle.Quad, Enum.EasingDirection.InOut)

Custom styles can be created for unique animations.

Examples

Full UI Implementation

Demonstrates a complete UI setup with introduction, watermark, notification, and components.

local XSX = loadstring(game:HttpGet('https://bauu.xyz/xsx.lua'))()
local Library = XSX:Init({
    title = "ExampleHook - Advanced",
    company = "ExampleStudio",
    Key = Enum.KeyCode.Insert,
    RainbowEnabled = true,
    BlurEffect = true,
    Debug = false
})

Library:BeginIntroduction()
Library:AddIntroductionMessage("Welcome to ExampleHook!")
Library:AddIntroductionMessage("Press Insert to toggle the interface.")
Library:AddIntroductionMessage("")
wait(2)
Library:EndIntroduction()

Library:Watermark("ExampleHook v1.0 | FPS: " .. Library.fps)

Library:Notify("UI Initialized Successfully!", 5, "success", function()
    print("notification triggered, you could do something else here, like auto load configs!")
end)

local MainTab = Library:NewTab("Main")
MainTab:NewSection("Player Controls")
MainTab:NewLabel("Adjust your character settings below", "center")
MainTab:NewButton("Reset Character", function()
    game.Players.LocalPlayer.Character:BreakJoints()
    Library:Notify("Character reset!", 3, "alert")
end)
MainTab:NewToggle("Infinite Jump", false, function(state)
    print("Infinite Jump: " .. tostring(state))
end):AddKeybind(Enum.KeyCode.J)
MainTab:NewSlider("Walk Speed", " studs/s", true, "/", {
    min = 16,
    max = 100,
    default = 16
}, function(value)
    game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = value
end)
MainTab:NewSelector("Team", "Neutral", {"Red", "Blue", "Neutral"}, function(team)
    print("Selected team: " .. team)
end)
MainTab:NewTextbox("Chat Message", "", "Enter message...", "medium", true, false, function(text)
    game.ReplicatedStorage.DefaultChatSystemChatEvents.SayMessageRequest:FireServer(text, "All")
end)
MainTab:NewSeperator()

local SettingsTab = Library:NewTab("Settings")
SettingsTab:NewSection("Game Settings")
SettingsTab:NewKeybind("UI Bind", Enum.KeyCode.Insert, function(key)
    Library:ShowUI(not Library.background.Visible)
end)

Dynamic Watermark Update

Updates the watermark text dynamically to display FPS.

local Watermark = Library:Watermark("Loading...")
game:GetService("RunService").RenderStepped:Connect(function()
    Watermark:SetText("Example UI v1.0.0 | FPS: " .. Library.fps)
end)

Troubleshooting

Common issues and solutions when using XSX.

  • UI not toggling: Ensure the keybind is not conflicting with other scripts. Use :SetKeybind(new) to change it.
  • Blur effect not working: Check if BlurEffect is enabled and BlurSize is set appropriately.
  • Script errors: Enable Debug = true in the config to log detailed errors to the console.
